# Cursor Clone - HTML/CSS Assignment

Preview Link : https://cursorvibe.vercel.app/

This project is a pixel-perfect recreation of the Cursor.com landing page, focusing on modern web design principles, responsive layout, and dark mode aesthetics.

## Recreated Sections

The following sections from the original Cursor website have been faithfully recreated:

1.  **Header & Navigation**: Sticky navbar with logo, navigation links, and action buttons.
2.  **Hero Section**: Main headline, subheadline, download CTA, and hero image.
3.  **Trusted By**: Grid of client logos (Trusted by millions...).
4.  **Feature Spotlights**:
    - "Agent" (Left-aligned text, Right image)
    - "Autocomplete" (Right-aligned text, Left image)
    - "Everywhere" (Left-aligned text, Right image)
5.  **Testimonials Grid / Feature Cards**: "The new way to build software" section with user quotes and profiles (Grid layout).
6.  **Capabilities / Testimonials**: "Stay on the frontier" section highlighting specific features (Models, Context, Security).
7.  **Changelog**: Timeline-style updates list.
8.  **About Team**: "Applied research team" section.
9.  **Recent Highlights**: News and blog posts section.
10. **Final CTA**: "Try Cursor now" bottom call-to-action.
11. **Footer**: Comprehensive 5-column footer with links, copyright, SOC 2 badge, theme toggles, and language selector.

## Design System

### Fonts

The project uses a combination of sans-serif for UI and monospace for code-related vibes:

*   **Primary Font**: `Inter` (Google Fonts) - Used for headings, body text, and UI elements.
*   **Code Font**: `IBM Plex Mono` (Google Fonts) - Used for specific technical details or code snippets (if applicable).

### Color Palette

A sophisticated dark mode palette was used to match the original "Cursor Vibe":

**Backgrounds**
*   `#14120B` (Primary Background - Deep Dark)
*   `#1B1913` (Secondary Background - Slightly Lighter for cards/sections)
*   `#CCCCCC` (Secondary Background Light)

**Typography**
*   `#FFFFFF` (Primary Text - Stark White)
*   `#9A9996` (Secondary Text - Muted Grey)
*   `#000000` (Black - Used on light buttons)

**Accents & Interactions**
*   `#f54e00` (Brand Orange - Links and Highlights)
*   `#BF4105` (Brand Orange Fade - Hover states)
*   `#45433E` (Borders and subtle architectural lines)

## Technologies

*   **HTML5**: Semantic structure.
*   **CSS3**: Custom properties (variables), Flexbox, CSS Grid, and responsive media queries.
*   **Remix Icon**: For UI icons (arrows, social logos, theme toggles).
